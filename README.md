## Интеллектуальная система AMWPro для постоянного сбора и анализа видео из социальных сетей на предмет мошеннических схем(игорный бизнес, финансовые пирамиды и др.) с применением моделей распознования речи, компьютерного зрения и генеративного искуственного интеллекта.
